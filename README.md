# TimeWaves — Sitio Web

Sitio oficial de TimeWaves, estudio de producción cinematográfica.

🌐 **Live:** https://timewaveslab.github.io/timewaveswebsite/

## Estructura

```
timewaves-website/
├── index.html              # Página principal
├── assets/
│   ├── images/             # Imágenes del sitio
│   │   ├── logo.png
│   │   ├── mascot-astronaut.png
│   │   └── mascot-monkey.png
│   └── video/              # Reel (próximamente)
├── raw-assets/             # Archivos originales (no se suben)
├── .gitignore
└── README.md
```

## Agregar el reel de video

1. Coloca el archivo de video en `assets/video/reel.mp4`
2. Descomenta la línea en `index.html`:
   ```html
   <source src="assets/video/reel.mp4" type="video/mp4">
   ```
